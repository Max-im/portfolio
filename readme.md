## **Setup the App**

- Download this repo:
  `git clone https://github.com/Max-im/portfolio`
- Go to the project directory:
  `cd PATH`
- Install all App dependencies:
  `npm run in`

## **Run the App**

- for start the project in develop mode:  
  `npm run dev`
